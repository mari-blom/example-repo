# example-repo
Project name: finance_calculators

Program that allows the user to access two different financial calculators: an investment calculator and a home loan repayment calculator.

The user is allowed to choose which calculation they want to do.

If the user selects “investment”, the user is asked to input:
○ The amount of money that they are depositing.
○ The interest rate (as a percentage). Only the number of the interest rate should be entered.
○ The number of years they plan on investing.
○ Then the user should input if they want “simple” or “compound” interest. 
Depending on their input, the output will show the appropriate amount that they will get back after the given period at the specified interest rate.

If the user selects “bond”, the user is asked to input:
○ The present value of the house.
○ The interest rate (as a percentage). Only the number of the interest rate should be entered.
○ The number of months they plan to take to repay the bond.
The output will show how much money the user will have to repay each month.
